Тема: Работа с GitBash

Некоторые команды:
	ls — просмотр содержимого директории
	cd — сменить директорию
		cd .. — поднятся на одну директорию вверх
	mkdir — создать директорию
	rm — удалить файл
		rm -r — удалить директорию
	touch — создать файл
	nano — консольный текстовый редактор

Команды с git:
	git clone — склонировать репозиторий (нужно указать ссылку GitHub на него)
	git status — просмотр изменений
	git add — добавить в commit файлы
		git add . — добавить все возможные файлы
	git commit -m "" — сделать commit, в кавычках пишется комментарий
	git push — отправить commit на репозиторий на GitHub
	git pull — перекачивает новые commit'ы с репозитория
	git init — инициализация пустого репозитория
	
	[название команды] -h — выводит возможные команды
